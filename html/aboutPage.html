Project Details

About

Pucalka is a small, proof-of-concept, collaborative effort by Team Coffee. The game is created for a teamwork assignment in the JavaScript DOM & UI course of Telerik Academy.

Concept

The game is inspired by an old Konami arcade game released in 1988 called Gradius II and known also as Vulcan Venture outside Japan. The game is a horizontally-scrolling shooter where the player can improve its ship with different powerups and fight a variety of enemies and bosses. More information is available on Wikipedia (https://en.wikipedia.org/wiki/Gradius_II)
